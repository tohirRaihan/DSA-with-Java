### Time and Space Complexity of Stack using Array
| Stack               | Time Complexity | Space Complexity |
|---------------------|-----------------|------------------|
| Create Stack        | `O(1)`          | `O(n)`           |
| Push                | `O(1)`          | `O(1)`           |
| Pop                 | `O(1)`          | `O(1)`           |
| Peek                | `O(1)`          | `O(1)`           |
| isEmpty             | `O(1)`          | `O(1)`           |
| Delete Entire Stack | `O(1)`          | `O(1)`           |
